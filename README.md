# Branch digital payment performance
Developed an end-to-end analytics solution using Excel and Power BI to monitor digital payment performance across multiple branches. The project involved creating and cleaning a simulated dataset containing daily transaction volumes, success rates, failure rates, and delayed payments across mobile money, card, and bank transfer channels. Interactive Power BI dashboards were built to compare branch performance, analyze payment trends, identify recurring failures, and highlight branches requiring operational attention through automated issue flagging.
